Instrucciones

Esta aplicación te permite explorar la relación entre las dos formas de expresar la ecuación de una recta:

  • Forma pendiente-intercepto: y = mx + b
  • Forma punto-pendiente: (y - y₁) = m(x - x₁)

Puedes:

  • Modificar la pendiente (m) y el intercepto (b) con los deslizadores
  • Arrastrar el punto rojo para cambiar el punto (x₁, y₁)
  • Ver el triángulo de pendiente que muestra gráficamente la relación Δy/Δx
  • Practicar con ejercicios en la pestaña "Ejercitar"
